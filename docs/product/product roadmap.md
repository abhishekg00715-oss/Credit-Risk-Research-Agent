**Phase 1**

Policy Research Assistant

Goal:
Policy Agent-
This agent will allow users to query credit policy documents.

Capabilities:

- PDF ingestion
- Embeddings
- Semantic search
- RAG responses

  
**Phase 2**

Customer Risk Assessment 

Goal:
Customer Risk Agent-
This agent will analyze customer credit profile.

Capabilities:

- Customer lookup
- Credit score evaluation
- Utilization analysis

**Phase 3**
Portfolio Intelligence

Goal:
Portfolio Agent-
Analyze historical portfolio performance.

Capabilities:

- Segment analysis
- Default rate analysis
- Risk trend detection

**Phase 4**

Recommendation Engine

Goal:
Recommendation Agent - this agent will combine findings from multiple agents.

**Phase 5**

Multi-Agent Workflow

Goal:

To integrate all the agents working together in an End-to-end orchestration.

Policy Agent
+
Customer Agent
+
Portfolio Agent
↓
Recommendation Agent
